<h1>Getting Started</h1>

<p>This section provides step-by-step instructions on how to install and configure the LabVIEW Computer Vision toolkit.</p>

<h2>To begin, please follow these steps</h2>

<h3>Install SOTA</h3>

<ol>
<li><a href="https://graiphic.io/download/"><strong>Download SOTA</strong></a>: In order to install the LabVIEW Computer Vision toolkit, you will first need to download SOTA. You can download SOTA by visiting the following link: <a href="https://graiphic.io/download/">https://graiphic.io/download/</a></li>
<li>Install SOTA: Once you have downloaded the SOTA setup file, run the installer and follow the on-screen instructions to install SOTA on your system.</li>
<li>Launch SOTA : After the installation is complete, launch SOTA from your desktop or start menu.</li>
</ol>

<p><a href="https://www.youtube.com/embed/4LJfdXmnj2w?feature=oembed">Get started - SOTA installation guide</a></p>

<h3>Install the Computer Vision Toolkit</h3>

<ol>
<li>Access LabVIEW Computer Vision Module: In the SOTA interface, locate and click on the LabVIEW Computer Vision module.</li>
<li>Select Version and Install: Select your LabVIEW version and click on the install button.</li>
<li>Accept License Agreement: Accept the terms of the license agreement to install the toolkit.</li>
<li>Launch LabVIEW: Once the installation is complete, you can launch LabVIEW and start utilizing theComputer Vision toolkit.</li>
</ol>

<p><strong>Congratulations! You have successfully installed the Compuer Vision toolkit using SOTA, and now you can make the most of your computer vision toolkit within LabVIEW.</strong></p>

<p>Note: Installing the Computer Vision Toolkit requires SOTA to be installed first. SOTA provides the interface to select, install, and activate the Computer Vision Toolkit within your LabVIEW environment.</p>

<p>If you encounter any issues or have any questions during the installation process, please refer reach out to our support team for assistance on <a href="https://graiphic.io/support-community/">support community page</a>.</p>

<p>We appreciate your interest in the Computer Vision toolkit and hope you find this installation guide helpful.</p>
