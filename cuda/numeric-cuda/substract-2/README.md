<h1>Substract</h1>

<h2>Description</h2>

<p>Computes the difference of the inputs. Type : <em><strong>polymorphic</strong><strong>.</strong></em></p>

<p><strong>Warning : A new tensor is created for the output if you substract two arrays together.</strong></p>
