<h1>FREQUENTLY ASKED QUESTIONS</h1>

<h2>Which LabVIEW functionalities are supported by the toolkit ?</h2>

<p>Perrine try to offers all existing LabVIEW functionalities and is constantly updated. The detail of the available functionalities is available <a href="https://graiphic.io/support-community/forum/official/releases/">here</a>.</p>

<h2>Does the toolkit support evaluation ?</h2>

<p>No, for now this toolkit is free.</p>

<h2>What type of hardware is supported by the toolkit ?</h2>

<p>Cuda Toolkit supports x86 based PC Windows OS.</p>

<h2>Can a user suggest adding a new feature?</h2>

<p>Yes, absolutely! We encourage user feedback and suggestions. Our teams are dynamic and efficient, and they will be more than happy to consider and add new functionalities. Don’t hesitate to share your ideas with us!</p>

<h2>Which version of LabVIEW is supported by the toolkit ?</h2>

<p>The latest version of the toolkit supports 64-bit versions of LabVIEW  2020 and LabVIEW 2025. For a specific version please <a href="https://perrine.graiphic.io/contact/">contact us</a>.</p>
