<p>In this section you’ll find a list of all CuBLAS fonctionalities.</p>

|  | **ICONS** | **DESCRIPTION** |
| --- | --- | --- |
| [amax](../amax-2/README.md) | ![Amax.Png](assets/amax.png) | This function finds the (smallest) index of the element of the maximum magnitude. |
| [amin](../amin/README.md) | ![Amin.Png](assets/amin.png) | This function finds the (smallest) index of the element of the minimum magnitude. |
| [asum](../asum-2/README.md) | ![Asum.Png](assets/asum.png) | This function computes the sum of the absolute values of the elements of vector x. |
| [axpy](../axpy-2/README.md) | ![Axpy.Png](assets/axpy.png) | This function multiplies the vector x by the scalar alpha and adds it to the vector y overwriting the latest vector with the result. |
| [copy](../copy/README.md) | ![Copy.Png](assets/copy.png) | This function copies the vector x into the vector y. |
| [dot](../dot-2/README.md) | ![Dot.Png](assets/dot.png) | This function computes the dot product of vectors x and y. |
| [nrm2](../nrm2-2/README.md) | ![Nrm2.Png](assets/nrm2.png) | This function computes the Euclidean norm of the vector x. |
| [rot](../rot-2/README.md) | ![Rot.Png](assets/rot.png) | This function applies Givens rotation matrix. |
| [rotg](../rotg-2/README.md) | ![Rotg.Png](assets/rotg.png) | This function constructs the Givens rotation matrix. |
| [rotm](../rotm-2/README.md) | ![Rotm.Png](assets/rotm.png) | This function applies the modified Givens transformation to vectors x and y. |
| [rotmg](../rotmg-2/README.md) | ![Rotmg.Png](assets/rotmg.png) | This function constructs the modified Givens transformation. |
| [scal](../scal-2/README.md) | ![Scal.Png](assets/scal.png) | This function scales the vector x by the scalar alpha and overwrites it with the result. |
| [swap](../swap-2/README.md) | ![Swap.Png](assets/swap.png) | This function interchanges the elements of vector x and y. |
| [gemv](../gemv-2/README.md) | ![Gemv.Png](assets/gemv.png) | This function performs the matrix-vector multiplication : y = α * op(A)x + β * y |
| [gemm](../gemm/README.md) | ![Gemm.Png](assets/gemm.png) | This function performs the matrix-matrix multiplication : C = α * op(A) * op(B) + β * C |
