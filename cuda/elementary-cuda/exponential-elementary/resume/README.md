<p>In this section you’ll find a list of all exponential fonctionalities.</p>

|  | **ICONS** | **DESCRIPTION** |
| --- | --- | --- |
| [Exponential](../exponential/README.md) | ![Exponential.Png](assets/exponential.png) | Computes the value of e raised to the x power, or the exponential of x. |
| [Exponential (Arg) – 1](../exponential-arg-1/README.md) | ![Exponential_Arg_Minus_One.Png](assets/exponential_arg_minus_one.png) | Computes 1 less than the value of e raised to the x power. |
| [Power Of 10](../power-of-10/README.md) | ![Power_Of_10.Png](assets/power_of_10.png) | Computes 10 raised to the x power. |
| [Power Of 2](../power-of-2/README.md) | ![Power_Of_2.Png](assets/power_of_2.png) | Computes 2 raised to the x power. |
| [Power Of X](../power-of-x/README.md) | ![Power_Of_X.Png](assets/power_of_x.png) | Computes x raised to the y power. |
| [Y-th Root of X](../y-th-root-of-x/README.md) | ![Y_Th_Root_Of_X.Png](assets/y_th_root_of_x.png) | Returns the yth root of the input value x. |
| [Natural Logarithm](../natural-logarithm/README.md) | ![Natural_Logarithm.Png](assets/natural_logarithm.png) | Computes the base e natural logarithm of x. |
| [Natural Logarithm (Arg + 1)](../natural-logarithm-arg/README.md) | ![Natural_Logarithm_Arg_Plus_One.Png](assets/natural_logarithm_arg_plus_one.png) | Computes the natural logarithm of (x + 1). |
| [Logarithm Base 10](../logarithm-base-2/README.md) | ![Logarithm_Base_10.Png](assets/logarithm_base_10.png) | Computes the base 10 logarithm of x. |
| [Logarithm Base 2](../logarithm-base-2-2/README.md) | ![Logarithm_Base_2.Png](assets/logarithm_base_2.png) | Computes the base 2 logarithm of x. |
| [Logarithm Base X](../logarithm-base-x/README.md) | ![Logarithm_Base_X.Png](assets/logarithm_base_x.png) | Computes the base x logarithm of y. |
