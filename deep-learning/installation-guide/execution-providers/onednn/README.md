<h1>LabVIEW OneDNN Driver</h1>

<p>This section explains how to install and configure OneDNN for the LabVIEW Deep Learning toolkit.</p>

<h2>System Requirements</h2>

<p>The LabVIEW Deep Learning toolkit run actually OneDNN version. To know if your GPU is compatible with this version, please consult<a href="https://gist.github.com/standaloneSA/99788f30466516dbcc00338b36ad5acf"> this page</a>.</p>

<h2>How to install the LabVIEW Deep Learning toolkit GPU INSTALLER ?</h2>

<p>To leverage GPU acceleration and fully utilize your OneDNN -compatible hardware in an optimized manner, you need to install OneDNN exclusively using SOTA.</p>

<ol>
<li>Launch SOTA: Launch SOTA..</li>
<li>Access OneDNN Section: In the SOTA interface, locate and click on Hardware then Intel – OneDNN in the top menu. This will take you to the OneDNN section.</li>
<li>Select Version and Install: Select your LabVIEW version and click on the install button.</li>
<li>Toolkit Ready: Congratulations, you’re now ready to run Graiphic toolkits on your GPU.</li>
</ol>
