<h1>LabVIEW CUDA Driver</h1>

<p>This section explains how to install and configure Cuda for the LabVIEW Deep Learning toolkit.</p>

<h2>System Requirements</h2>

<p>The LabVIEW Deep Learning toolkit run actually CUDA 12.5 version. To know if your GPU is compatible with this version, please consult<a href="https://gist.github.com/standaloneSA/99788f30466516dbcc00338b36ad5acf"> this page</a>.</p>

<h2>How to install the LabVIEW Deep Learning toolkit GPU INSTALLER ?</h2>

<p>To leverage GPU acceleration and fully utilize your CUDA-compatible hardware in an optimized manner, you need to install CUDA exclusively using SOTA.</p>

<ol>
<li>Launch SOTA: Launch SOTA..</li>
<li>Access CUDA Section: In the SOTA interface, locate and click on Hardware then Nvidia – CUDA in the top menu. This will take you to the CUDA section.</li>
<li>Select Version and Install: Select your LabVIEW version and click on the install button.</li>
<li>Toolkit Ready: Congratulations, you’re now ready to run Graiphic toolkits on your GPU.</li>
</ol>
