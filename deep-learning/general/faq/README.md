<h1>FREQUENTLY ASKED QUESTIONS</h1>

<h2>Which layers are supported by the toolkit ?</h2>

<p>HAIBAL offers all existing layers and is constantly updated. The detail of the available layers is available <a href="https://haibal.com/support-community/forum/official/releases/">here</a>.</p>

<h2>Do we need Python Language installation to run the toolkit ?</h2>

<p>No. HAIBAL deep learning toolkit is fully powered in LabVIEW. We worked more than 2 years to rewrite all the layers and functionalities in native LabVIEW.</p>

<h2>Does the toolkit support evaluation ?</h2>

<p>Yes, you will get 30 days evaluation period after the installation.</p>

<h2>How many simultaneous installation can i do with a licence ?</h2>

<p>We have the same policy as NI with LabVIEW  on licencing. The licence is for one user and allow to install on 3 computer simultaneously.</p>

<h2>How to move already activated license to another computer ?</h2>

<p>You will need to deactivate the toolkit on the PC where it is already activated and activate the same license on the other PC.</p>

<h2>What type of hardware is supported by the toolkit ?</h2>

<p>HAIBAL supports x86 based PC Windows OS for training and inference (a Linux version is under development).<br/>
FPGAs, AMD GPUs, intel GPUs will be integrate in the future (also under development).</p>

<h2>Does the toolkit support GPU acceleration ?</h2>

<p>Yes, the training and inferences can be accelerated on Nvidia GPUs, <a href="../../installation-guide/execution-providers/cuda/README.md">more information</a>.</p>

<h2>Can a user suggest adding a new feature?</h2>

<p>Yes, absolutely! We encourage user feedback and suggestions. Our teams are dynamic and efficient, and they will be more than happy to consider and add new functionalities. Don’t hesitate to share your ideas with us!</p>

<h2>Which version of LabVIEW is supported by the toolkit ?</h2>

<p>The latest version of the toolkit supports 64-bit versions of LabVIEW  2020 and LabVIEW 2023. For a specific version please <a href="https://haibal.com/contact/">contact us</a>.</p>

<p>You can contact us directly <a href="mailto:contact@haibal.com">by mail</a> or on this <a href="https://haibal.com/contact/">form page</a>.</p>
