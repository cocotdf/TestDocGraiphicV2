<h2>Dimension</h2>

<h3>Input shape</h3>

<p>All layer used for concatenation must have the same shape, except along the concatenation axis.<br/>Refer to each layer’s output shape and ensure compatibility on all axes except the concatenation one.</p>

<h3>Output shape</h3>

<p>The output shape is identical to the input shape, except on the concatenation axis, where the sizes are summed.</p>

<h2>Example</h2>

<p>All these exemples are snippets PNG, you can drop these Snippet onto the block diagram and get the depicted code added to your VI (Do not forget to install Deep Learning library to run it).</p>

<h3>Concatenate layer with two identical input layer shape</h3>

<p align="center"><img src="assets/1-concat-with-two-identical-input-layer-shape.png" alt="1 - Concat with two identical input layer shape" width="260" /></p>

<p>1 – Generate a set of data</p>

<p>We generate an array of data of type single and shape [batch_size = 10, input_dim = 5] (same input shape).</p>

<p>2 – Define graph</p>

<p>We first define two input layers named input_dense1 and input_dense2. This layers is setup as an input array shaped [input_dim = 5].<br/>In order to have same output shape for added dense layers we define for both of these the same “units” parameter (units = 5) (refer <a href="../dense-add-to-graph/README.md">Dense</a> layer add to graph documentation for more details).<br/>Then we build an array of the two graphs generated by the dense layers and inject it into the input of the Concatenate layer.</p>

<p>3 – Summarize graph</p>

<p>Returns the summary of the model in file text.</p>

<p>3 – Run graph</p>

<p>We call the forward method and retrieve the result with the “Prediction 2D” method.<br/>This method returns two variables, the first one is the layer information (cluster composed of the layer name, the graph index and the shape of the output layer) and the second one is the prediction with a shape of [batch_size, units] (Dense output shape).</p>
