<h1>QLinearConcat</h1>

<h2>Description</h2>

<p>Concatenate a list of tensors into a single tensor.All input tensors must have the same shape, except for the dimension size of the axis to concatenate on.</p>

<h2>Type Constraints</h2>

<p><strong>T8</strong> in (<code>tensor(uint8)</code>, <code>tensor(int8)</code>) : Constrain input and output types to 8 bit signed and unsigned tensors.</p>

<p><strong>TF</strong> in (<code>tensor(float)</code>) : Constrain scale types to any float tensor type.</p>

<p><strong>TV</strong> in (<code>tensor(uint8)</code>, <code>tensor(int8)</code>, <code>tensor(float)</code>) : Sequence of (Tensor, Scale, ZeroPoint) tuples. The type is sequence of (T8, TF, T8).</p>
