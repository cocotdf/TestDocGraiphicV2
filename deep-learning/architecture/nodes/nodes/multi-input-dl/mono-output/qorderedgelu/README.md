<h2>Type Constraints</h2>

<p><strong>Q</strong> in (<code>tensor(int8)</code>) : Constrain input and output types to int8 tensors.</p>

<p><b>S </b>in (<code>tensor(float)</code>) : Constrain scales to float32.</p>
