<h1>RotaryEmbedding</h1>

<h2>Description</h2>

<p>RotaryEmbedding is the implementation of rotary positional embeddings (RoPE). The positions are represented as rotation matrices that are multiplied to query and key before the inner product of query and key is taken.</p>

<h2>Type Constraints</h2>

<p><strong>T</strong> in (<code>tensor(float)</code>, <code>tensor(float16)</code>, <code>tensor(bfloat16)</code>) : Constrain input and output types to float tensors.</p>

<p><strong>M</strong> in (<code>tensor(int64)</code>) : Constrain input and output types to integer tensors.</p>
