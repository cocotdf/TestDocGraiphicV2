<h2>Type Constraints</h2>

<p><strong>T</strong> in (<code>tensor(float16)</code>) : Constrain input and output types to float16 tensors.</p>

<p><strong>U</strong> in (<code>tensor(float)</code>) : Constrain input 0 type to float tensors.</p>
