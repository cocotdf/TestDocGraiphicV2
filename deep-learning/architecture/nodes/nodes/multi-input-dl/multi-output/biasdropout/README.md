<h1>BiasDropout</h1>

<h2>Description</h2>

<p>output, dropout_mask = Dropout(data + bias, ratio) + residual, Intended to specialize the dropout pattern commonly found in transformer models.</p>

<h2>Type Constraints</h2>

<p><strong>T</strong> in (<code>tensor(bfloat16)</code>, <code>tensor(double)</code>, <code>tensor(float)</code>, <code>tensor(float16)</code>) : Constrain input and output types to float tensors.</p>

<p><strong>T1</strong> in (<code>tensor(bfloat16)</code>, <code>tensor(double)</code>, <code>tensor(float)</code>, <code>tensor(float16)</code>) : Constrain input ‘ratio’ types to float tensors.</p>

<p><strong>T2</strong> in (<code>tensor(bool)</code>) : Constrain output ‘mask’ types to boolean tensors.</p>
