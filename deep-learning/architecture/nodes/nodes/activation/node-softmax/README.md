<h1>Softmax</h1>

<h2>Description</h2>

<p>The operator computes the normalized exponential values for the given input : Softmax(input, axis) = Exp(input) / ReduceSum(Exp(input), axis=axis, keepdims=1)<br/>The “axis” attribute indicates the dimension along which Softmax will be performed. The output tensor has the same shape and contains the Softmax values of the corresponding input.</p>
