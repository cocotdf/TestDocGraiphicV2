<h1>Celu</h1>

<h2>Description</h2>

<p>Continuously Differentiable Exponential Linear Units: Perform the linear unit element-wise on the input tensor X using formula : max(0,x) + min(0,alpha*(exp(x/alpha)–1))</p>
