<div><h2>Get parameters from model</h2></div>
</div><div>
<div><p>Color used for the family of functionality that get parameters from model.<br/>Inside this family there is three distinct group as “<strong>Get All</strong>“, “<strong>Get by Name</strong>” and “<strong>Get by Index</strong>“.</p>
<ul>
<li>“<strong>Get All</strong>” will give All layers choosen parameter.</li>
<li>“<strong>Get by Name</strong>” will give by layer name the choosen parameter.</li>
<li>“<strong>Get by Index</strong>” will give by layer Index the choosen parameter.</li>
</ul>

<div><h2>Set parameters from model</h2></div>
</div><div>
<div><p>Color used for the family of functionality that set parameters of the model.<br/>Inside this family there is three distinct group as “<strong>Set All</strong>“, “<strong>Set by Name</strong>” and “<strong>Set by Index</strong>“.</p>
<ul>
<li>“<strong>Set All</strong>” will set choosen parameter for All layers.</li>
<li>“<strong>Set by Name</strong>” will set the choosen parameter by the Name of the layer.</li>
<li>“<strong>Set by Index</strong>” will set the choosen parameter by the Index of the layer.</li>
</ul>
