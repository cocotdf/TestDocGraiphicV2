<h1>Examples Guide</h1>

<h2>How to run online examples ?</h2>

<p>All online example present in this documentation are snippets PNG, you can drop these Snippet onto the block diagram and get the depicted code added to your VI (Do not forget to install the LabVIEW deep learning toolkits library to run it).</p>

<p><a href="https://www.youtube.com/embed/lQMGqAvbOKI?feature=oembed">Get Started - Save model guide</a></p>

<ul>
<li>
<p>Go to the LabVIEW deep learning palette dans launch example.</p>
</li>
</ul>

<p><a href="https://www.youtube.com/embed/7uxPpuBK0K8?feature=oembed">Get Started - Deep learning toolkit quick launch of examples from the palette</a></p>

<h2>How to launch documentation guide inside LabVIEW ?</h2>

<ul>
<li>
<p>Open<strong> Online documentation</strong> shortcut from “Help&gt;&gt;Graiphic&gt;&gt;Deep Learning&gt;&gt;” menu from LabVIEW.</p>
</li>
</ul>

<p><a href="https://www.youtube.com/embed/ZETvhKJZ4yU?feature=oembed">How to launch the LabVIEW deep learning online documentation from LabVIEW</a></p>

<h2>Accurate documentation help from VI ?</h2>

<ul>
<li>Open<strong> Context HELP</strong> shortcut from “Help” menu from LabVIEW (ore use keyword CTRL+H shortcut).</li>
</ul>
