<h1>Getting Started</h1>

<h3>Install the Online Accelerator Toolkit</h3>

<ol>
<li>Access LabVIEW Accelerator Module: In the SOTA interface, locate and click on the LabVIEW Accelerator module.</li>
<li>Select Version and Install: Select your LabVIEW version and click on the install button.</li>
<li>Accept License Agreement: Accept the terms of the license agreement to install the toolkit.</li>
<li>Launch LabVIEW: Once the installation is complete, you can launch LabVIEW and start utilizing the LabVIEW Accelerator toolkit.</li>
</ol>

<p><strong>Congratulations! You have successfully installed the Accelerator toolkit using SOTA, and now you can make the most of your Accelerator toolkit within LabVIEW.</strong></p>

<p>Note: Installing the LabVIEW Accelerator Toolkit requires SOTA to be installed first. SOTA provides the interface to select, install, and activate the LabVIEW Accelerator Toolkit within your LabVIEW environment.</p>

<p>If you encounter any issues or have any questions during the installation process, please refer reach out to our support team for assistance on <a href="https://graiphic.io/support-community/">support community page</a>.</p>

<p>We appreciate your interest in the LabVIEW Accelerator toolkit and hope you find this installation guide helpful.</p>

<p><a href="https://www.youtube.com/embed/2SS5SrcWxe0?feature=oembed">Get started - Accelerator installation guide</a></p>
